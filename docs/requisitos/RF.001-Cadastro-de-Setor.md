# Cadastro de Setor

**COMO** Administrador da Clínica,

**QUERO** cadastrar um novo setor,

**PARA** organizar os ambientes da clínica veterinária.

## Critérios de Aceitação

- Deve ser possível informar o nome do setor.
- O nome do setor deve ser obrigatório.
- O sistema deve impedir setores com nomes duplicados.
- O setor deve ser salvo no banco de dados.
