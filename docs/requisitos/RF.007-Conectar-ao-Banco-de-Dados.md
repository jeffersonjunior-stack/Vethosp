# Conectar ao Banco de Dados

**COMO** Sistema,

**QUERO** estabelecer conexão com o banco de dados,

**PARA** armazenar e recuperar as informações da clínica.

## Critérios de Aceitação

- O sistema deve conectar ao banco MySQL.
- Caso a conexão falhe, uma mensagem de erro deve ser apresentada.
- A conexão deve ser utilizada para todas as operações de cadastro e consulta.
