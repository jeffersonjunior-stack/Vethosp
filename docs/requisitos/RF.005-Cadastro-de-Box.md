# Cadastro de Box

**COMO** Administrador da Clínica,

**QUERO** cadastrar um novo box,

**PARA** organizar os espaços destinados aos animais.

## Critérios de Aceitação

- Deve ser possível informar o nome do box.
- Deve ser possível selecionar o setor ao qual o box pertence.
- O sistema deve salvar o box no banco de dados.
