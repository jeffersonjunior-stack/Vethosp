# Exclusão de Box

**COMO** Administrador da Clínica,

**QUERO** excluir um box,

**PARA** remover registros que não são mais utilizados.

## Critérios de Aceitação

- O sistema deve solicitar confirmação antes da exclusão.
- O box deve ser removido do banco de dados quando permitido.
- O sistema deve informar o resultado da operação.
