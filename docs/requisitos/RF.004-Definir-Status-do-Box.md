# Definir Status do Box

**COMO** Administrador da Clínica,

**QUERO** definir o status de um box,

**PARA** informar sua disponibilidade para atendimento.

## Critérios de Aceitação

- Deve ser possível definir o status do box.
- O sistema deve aceitar apenas os status cadastrados.
- O status deve ser salvo no banco de dados.
- O status deve ser exibido na listagem dos boxes.
