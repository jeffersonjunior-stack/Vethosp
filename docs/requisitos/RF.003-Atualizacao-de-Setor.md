# Atualização de Setor

**COMO** Administrador da Clínica,

**QUERO** atualizar as informações de um setor,

**PARA** manter os dados corretos.

## Critérios de Aceitação

- Deve ser possível editar o nome do setor.
- As alterações devem ser salvas no banco de dados.
- O sistema deve informar quando a atualização for concluída.
