# Gerenciar Dados da Clínica

**COMO** Administrador da Clínica,

**QUERO** gerenciar os setores e os boxes,

**PARA** manter as informações da clínica organizadas e atualizadas.

## Critérios de Aceitação

- Deve ser possível cadastrar setores.
- Deve ser possível cadastrar boxes.
- Deve ser possível visualizar os dados cadastrados.
- O sistema deve manter as informações persistidas no banco de dados.
