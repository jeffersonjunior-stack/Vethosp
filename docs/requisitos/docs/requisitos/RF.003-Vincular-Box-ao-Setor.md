# Vincular Box ao Setor

**COMO** Administrador da Clínica,

**QUERO** associar um box a um setor,

**PARA** organizar corretamente os espaços de atendimento da clínica.

## Critérios de Aceitação

- Deve ser possível selecionar um setor já cadastrado.
- Cada box deve estar vinculado a apenas um setor.
- O sistema deve impedir o cadastro de um box sem um setor associado.
- O vínculo deve ser salvo no banco de dados.
