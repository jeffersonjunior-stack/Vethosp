# Atualização de Box

**COMO** Administrador da Clínica,

**QUERO** atualizar as informações de um box,

**PARA** manter seus dados corretos.

## Critérios de Aceitação

- Deve ser possível editar o nome do box.
- Deve ser possível alterar o setor ao qual o box pertence.
- As alterações devem ser salvas no banco de dados.
